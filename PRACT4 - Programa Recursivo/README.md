Ventajas de cada método

Característica            |          Solución Iterativa                             |         SoluciónRecursiva
--------------------------------------------------------------------------------------------------------------------------------
Eficiencia                |      Muy alta. Solo recorre el ciclo una vez ($O(n)$).  |     Muy baja. Recalcula los mismos valores muchas veces ($O(2^n)$).
                          |                                                         |
Memoria                   |     Usa poca memoria (variables simples).               |    Consume mucha memoria por la pila de llamadas (stack).
                          |                                                         |
Legibilidad               |      Puede ser un poco más extensa de escribir.         |     Muy elegante y similar a la definición matemática original.
                          |                                                         |
Recomendación             |      Uso profesional y producción.                      |     Fines académicos para entender la recursividad.
